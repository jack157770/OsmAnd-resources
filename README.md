OsmAnd-resources
================

Custom defined resources of the OsmAnd project to fine tune OsmAnd behavior.
Defined here are:
* map rendering styles used in OsmAnd
* recorded and TTS voices
* configuration of the offline map generation
* routing parameters
* icon repository
